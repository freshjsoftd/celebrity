Авторизация в GitHub через SSH key: 
Войти в Docs=>authentication=>connect with ssh=>Generate new SSH key=>copy ssh-keygen -t rsa -b 4096 -C "your_email@example.com"=> put it to terminal and press Enter.
Открыть папку home/user/.ssh
Открыть с помощью редактора файл id_rsa.pub
Скопировать весь ключ, от ssh-rsa до почты
Зайти в setting GitHub=>SSH and GPG keys=>New SSH key (зеленая кнопка вверху)=>ввести какое либо название в строке Title (на английском в одно слово)=>в поле Key вставить свой скопированный ключ=>press AddSSH key.
Теперь при создании нового репозитория в GitHub нужно будет выбирать не HTML а SSH. Копировать ссылку и дальше как обычно git remote add origin + скопированная ссылка. Дальше следовать указаниям. набрать yes  и т.д.
